# Asset slots

This demo currently renders simple pixel-style shapes in Canvas.

Reserved folders for later replacement:

- `characters/`: samurai player sprites and animation sheets.
- `enemies/`: goblin, wolf, and boss sprites.
- `backgrounds/`: sunset ruin backgrounds and parallax layers.
- `tiles/`: ground, platform, wall, and ruin tiles.
- `ui/`: icons, portraits, buttons, and HUD art.
